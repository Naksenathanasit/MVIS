## Getting Started

1. Clone the repo

   ```sh
   git clone https://github.com/mwit30-8/MVIS.git
   ```

2. Build docker image

   ```sh
   docker-compose build
   ```

3. Run the software at <http://localhost:19006>

   ```sh
   docker-compose up
   ```
